# Plan: Add `!mutatepool` Command for TMT Season 2

## Overview
Add a new command `!mutatepool <@[!]?player>` to the TMT league that:
1. Finds "Remaining from Pool Pending" entries for the player in Pool Changes
2. Mutates each of those pools
3. Generates 2 new unmutated TMT packs
4. Combines into a full season 2 starting pool
5. Writes to "S2 Pool Changes" sheet
6. Posts pool link + tiled rare image to Starting Pools channel

## Files to Modify

### 1. `leagues/tmt/pool-gen.ts` (~line 575)
Add new command handler `mutatepoolHandler`

### 2. `leagues/tmt/tmt.ts` (~line 1153)
Export new functions or modify setup to include the new handler

## Implementation Details

### Step 1: Parse Command
- Use existing `resolveDiscordId(parts[1])` from pool-gen.ts
- Look up player via `getPlayers()` using Discord ID
- Get `playerName = player.Identification`

### Step 2: Find Pool Changes with "Remaining from Pool Pending"
```typescript
const allChanges = await getPoolChanges();
const playerChanges = allChanges.rows.filter(
  (c) => c.Name === playerName && c.Comment === "Remaining from Pool Pending"
);
```
Each change has `Value` = sealeddeck pool ID.

### Step 3: Mutate Each Pool
For each pool ID in the changes:
```typescript
const pool = await fetchSealedDeck(poolId);
const { mutatedCards } = await mutateWholePool(pool);
```
- Mutation always succeeds (if no candidates found, returns original card - no failure case)
- Accumulate all mutated ScryfallCards (just for the tiled image later)

### Step 4: Generate 2 New TMT Packs
Use `waitForBoosterTutor` from pending.ts:
- Send `!TMT 2` to Bot Bunker (or starting pool channel)
- Wait for response using `waitForBoosterTutor`
- Parse response txt into 2 individual packs using same logic as `generateStartingPool` (`splitIntoPacks`)
- Create sealeddeck for each pack, store pool IDs

### Step 5: Combine into Full Pool
Combine all cards from:
- Mutated pools (from step 3)
- 2 new TMT packs (from step 4)

Create sealeddeck:
```typescript
const fullPoolId = await makeSealedDeck({ sideboard: combinedCards });
```

### Step 6: Write to "S2 Pool Changes" Sheet
Write all 7 entries in a single batch call using `addPoolChanges` with `sheetName="S2 Pool Changes"`:
- 4 "add pack" entries for mutated pools (Value = mutated pool ID, Comment = "Mutated from pool <original>")
- 2 "add pack" entries for new TMT packs (Value = new pack ID)
- 1 "starting pool" entry (Value = fullPoolId = full sealeddeck link ID, Comment = "Season 2 starting pool")

### Step 7: Post to Starting Pools Channel
Reply to the original command message:
1. Send pool link: `https://sealeddeck.tech/${fullPoolId}`
2. Generate tiled rare image:
   - Get all ScryfallCards from combined pool (both mutated and new TMT packs)
   - Look up each card name via `getAllArenaCards()` → `buildNameToCardMap()`
   - Filter for rares/mythics/special/bonus
   - Call `tileRareImages(rareCards, "small")`
3. Send as attachment with embed

## Registration
In `leagues/tmt/tmt.ts` `setup()`, add `mutatepoolHandler` to `messageHandlers` array.

## Edge Cases
- Player lookup fails → error message
- No "Remaining from Pool Pending" entries for player → error message
- Fetching a pool fails (invalid pool ID) → fail entire operation, do NOT give partial pool
- Booster Tutor timeout or fails to return 2 packs → fail entire operation
